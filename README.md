# NodeJS TypeScript Starter Template

## Installation
`` npm install typescript gulp -g & npm install ``

## Building

| Action        | Command       | Description  |
| ------------- |:-------------:| ------------:|
| Build         | gulp build    |              |
| clean         | gulp clean    |              |
| run           | gulp run      |              |
| run           | gulp test     |              |

## Included Modules
* Express
* MySQL

#### Testing
* Mocha
* Should

#### Build Tools
* gulp
* gulp-clean
* gulp-mocha
* gulp-run
* gulp-typescript
* typescript
